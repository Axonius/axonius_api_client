Select Fields: Invalid Adapter
###############################################

This does the following:

* Raises an error that shows all of the valid adapters because no adapter
  named ``foo`` exists.

.. code:: shell

   $ axonshell devices get \
     --field foo:bar \
     --export-file device.json \
     --export-overwrite
