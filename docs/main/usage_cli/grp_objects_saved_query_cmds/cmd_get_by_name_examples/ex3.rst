Multiple using Regular Expressions
###############################################

This does the following:

* Gets multiple saved queries that match a regular expression.

.. code:: shell

   $ axonshell devices saved-query get-by-name \
     --value "[y-z]" \
     --value-regex \
     --export-file saved_query.json \
     --export-overwrite
