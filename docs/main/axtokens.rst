API Key and Secret
#########################

.. include:: /main/deprecation_banner.rst

The API Key and Secret for an Axonius user is available from the :blue:`My Account` page.

You can get to the :blue:`My Account` page by clicking the gear icon in the left hand
toolbar or by going to :red:`https://axonius_instance/account`. On that page you will find
the :green:`API Key` and :green:`API Secret` fields under the :blue:`API Key` tab:

   :scale: 60
